# Merey Wedding Invitation

Static wedding invitation site prepared for GitHub Pages.

## Files

- `index.html` - main invitation page.
- `merey-music.mp3` - background music used by the page.
- `.nojekyll` - tells GitHub Pages to serve files as-is.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open repository settings.
4. Go to `Pages`.
5. In `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and `/root`, then save.

The site will be available at the GitHub Pages URL after deployment finishes.
